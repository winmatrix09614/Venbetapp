## 4. ТЕКСТ САМОГО ПРОГНОЗА (AI-разбор)
*Файл: `services.py` → функция `generate_prediction_text`, переменная `prompt`.*

Это НЕ фиксированный текст — его пишет ИИ (Gemini) по инструкции (промпту).
Чтобы изменить СТИЛЬ/ТОН разбора — редактируется промпт (блок «ТОН:», «НЕЛЬЗЯ:»,
«ПРИМЕР ТОНА»). Сейчас тон: «уверенный, живой, профессиональный каппер, 6–8
предложений, опора на цифры». Резервный текст (если ИИ не ответил) — в `_fb`.

## 5. АППКА (интерфейс) — кнопки, меню, подписи
*Файл: `src/themes.js` → блок `default` (ru). После правки ru — внести в es/pt/fr/tr/az/en.*

### Экран входа
- subtitle: Нейросеть для анализа спорта
- inputLabel: Ваш ID 1xBet
- inputPlaceholder: Например: 10000000
- inputDesc: ID можно найти в приложении 1xBet → Профиль
- btnText: Продолжить
- waitingTitle: Ожидание подтверждения
- waitingDesc: Ваш ID отправлен менеджеру. Дождитесь активации.
- idError: Неверный ID
- idLowNote: ID ниже 168 — старый аккаунт, точность прогнозов может быть ниже.
- howToFindId: Где взять ID?

### Главное меню (карточки)
- AI Анализ / Глубокий разбор матча
- История / Журнал сессий
- Сводка / Спортивные новости
- Поддержка / Связь со специалистом
- Прогноз дня / Бесплатный сигнал

### Промо-блок и «Как это работает»
- promoTitle: Сделай свой AI-прогноз
- promoSub: Узнай, кто победит — за секунды
- promoBtn: Начать
- hiwTitle: Как это работает
- hiwStep1: Введи 1xBet ID
- hiwStep2: AI анализирует
- hiwStep3: Ставишь увереннее

### Экран прогноза
- analysisInit: Система инициализирована. Загрузите скриншот матча или введите названия команд для начала AI анализа.
- verdictTag: ВЕРДИКТ НЕЙРОСЕТИ
- confidence: Уверенность
- baseConf: Базовая оценка
- liveBadge: 🔴 LIVE {score}, {minute}′ · прогноз предматчевый
- disclaimer: Это AI-аналитика, а не гарантия. Решение о ставке принимаете вы.
- singleNote: Это одиночная ставка — рекомендуем ставить отдельно.
- expressNote: Готовый экспресс — ставьте его целиком, как составлен.
- combined: Суммарная уверенность

### Статистика команд
- statBlockTitle: Статистика команд
- statForm: Форма / statGoalsFor: Голы в среднем / statGoalsAg: Пропускает
- statPoss: Владение % / statShots: Удары / statShotsOn: В створ
- statCorners: Угловые / statH2H: Очные встречи

### История
- historyTitle: ИСТОРИЯ / historyLoading: Загрузка архива...
- historyEmpty: История пуста / historyEmptySub: Вы еще не делали прогнозов
- archiveTag: АРХИВ / undefinedResult: Неопределено / draw: Ничья

### Новости/Сводка
- newsMainTag: ГЛАВНОЕ / newsLatestTag: Последние новости
- newsActualTag: АКТУАЛЬНОЕ / newsRead: Читать →
- newsUnavailable: Новости временно недоступны / newsTryLater: Попробуйте позже

### Прочее
- errorPrefix: Ошибка / connError: Ошибка соединения с сервером.
- loading: Загрузка приложения... / attemptsLeft: Осталось прогнозов
- single: Одиночный / express: Экспресс
- sportFootball: Футбол / sportBasketball: Баскетбол
- dailyTitle: Прогноз дня / dailyDesc: Бесплатный сигнал / dailyEmpty: Прогноз дня пока не задан
- statsCapper: Статистика прогнозов / predictionsWord: прогнозов / accuracyWord: точность / per7d: за 7 дней
- howToBet: Как поставить ставку / videoSoon: Обучающее видео скоро появится
- bannedTitle: Доступ заблокирован / logoutConfirm: Вы уверены, что хотите выйти?

## КАК ВНОСИТЬ ПРАВКИ (для разработчика)
1. Тимлид правит русские тексты в этом файле.
2. Разработчик переносит правки в код (русский), затем переводит на 6 языков.
3. Места: бот — `bot.py _BOT_MSG`; сводка — `retention.py`; ошибки/прогноз —
   `web/webapp.py`; стиль AI-разбора — `services.py` (промпт); аппка — `src/themes.js`.
